# 🐱 猫猫独立钻石棋 (Peg Solitaire)

一款可爱的像素风格益智游戏，使用萌萌的小猫作为棋子！

---

## 🎮 游戏玩法

### 基本规则

独立钻石棋是一款经典的单人益智游戏，目标是通过跳跃吃子，尽可能减少棋盘上剩余的棋子数量。

1. **跳跃移动**：选中一个棋子，让它跳过相邻的另一个棋子到达空位
2. **吃子规则**：被跳过的棋子会被移除（吃掉）
3. **移动方向**：只能横向或纵向跳跃，**不能斜跳**
4. **游戏目标**：剩余棋子越少越好，最理想是只剩1个且在中央

### 操作方法

| 操作 | 效果 |
|------|------|
| 点击棋子 | 选中该棋子（高亮显示） |
| 点击绿色高亮空位 | 跳跃到该位置 |
| 再次点击已选中的棋子 | 取消选中 |
| 点击另一个棋子 | 切换选中 |

### 评级系统

根据最终剩余的棋子数量，你将获得以下评级：

| 剩余棋子 | 评级 | 表现 |
|---------|------|------|
| 6个或以上 | 一般 | 继续加油！ |
| 5个 | 颇好 | 不错的尝试 |
| 4个 | 很好 | 表现优秀 |
| 3个 | 聪明 | 相当厉害 |
| 2个 | 尖子 | 非常出色 |
| 1个 | 大师 | 完美通关！ |
| 1个且在中央 | 天才 | 最高成就！ |

## 📜 游戏历史

独立钻石棋（Peg Solitaire）是一种起源于17世纪法国的单人棋盘游戏。传说是一位被关在法国巴士底狱的贵族发明的，用来打发狱中的漫长时光。

英式33孔棋盘是最经典的版本，棋盘呈十字形，共有33个孔位，初始状态下中心位置为空，其余32个位置都有棋子。

## 🎯 游戏技巧

1. **从边缘开始**：优先移动边缘的棋子，避免它们被困住
2. **保持对称**：尽量保持棋盘的对称性，有助于规划后续移动
3. **向中心收拢**：逐步将棋子向中心区域集中
4. **提前规划**：思考几步之后的局面，避免陷入死局

# Peg Solitaire (独立钻石棋) 🐱

一款使用 Godot 4.5 开发的 2D 像素风格 Peg Solitaire 游戏，棋子使用可爱的小猫素材！

## 游戏规则

游戏玩法类似中国跳棋，但只能跳不能走步：

1. **跳跃移动**: 棋子只能跳过相邻的棋子到空位上
2. **吃子规则**: 被跳过的棋子会被吃掉（移除）
3. **移动方向**: 棋子可以沿格线横、纵方向跳，但不能斜跳
4. **游戏目标**: 剩下越少棋子越好

## 评级系统

| 剩余棋子 | 评级 |
|---------|------|
| 6只或以上 | 一般 |
| 5只 | 颇好 |
| 4只 | 很好 |
| 3只 | 聪明 |
| 2只 | 尖子 |
| 1只 | 大师 |
| 1只且在正中央 | 天才 |

## 操作说明

1. **点击棋子** - 选中一个棋子（高亮显示）
2. **点击空位** - 如果可以跳跃，棋子会跳到该位置并吃掉中间的棋子
3. **再次点击选中的棋子** - 取消选中
4. **点击另一个棋子** - 切换选中

## 特色功能

- 🎨 **像素风格**: 使用可爱的像素小猫作为棋子
- 🎵 **音效系统**: 
  - 背景音乐：鸟鸣声
  - 点击棋子：猫叫声
  - 无效移动：猫嘶叫声
- ✨ **动画效果**: 棋子跳跃动画、被吃棋子消失动画
- 🎯 **英式33棋盘**: 经典的英式独立钻石棋盘布局

## 如何运行

1. 使用 Godot 4.5 打开项目
2. 按 F5 或点击运行按钮启动游戏

## 开发说明

- **引擎版本**: Godot 4.5
- **渲染器**: Forward+
- **分辨率**: 800x600
- **像素过滤**: Nearest (保持像素清晰)
